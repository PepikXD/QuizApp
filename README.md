# QuizApp
Simple quiz app where you can create and edit your quizzes

# How to run
Open project in IDE or code editor of your choice and run the project

# Requirements
Maven 4.0  
Java 22

# External libreries
[Lombok](https://projectlombok.org) 1.18.36
